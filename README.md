simgraph-explorer
=================

A [Shiny](http://shiny.rstudio.com/) app for exploring the many random graph generators in [igraph](http://igraph.org/r/).

To run:

```r
shiny::runGitHub("simgraph_explorer", "mdlincoln")
```
